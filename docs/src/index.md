# EEGToolkit.jl

*Computational EEG analysis with emphasis in sleep neuroscience.*

## Package Features
- Loading EEG data
- EEG visualization
- Sleep stage handling and NREM period detection
- Power spectral analysis
- Spindle detection algorithms
